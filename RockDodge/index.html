Rock Dodge

Project information

  • Project date 12 May, 2023
  • Github Rock Dodge

Description

Rock dodge was a fun project to work on while learning coding basics. The game is a simple 2D game where you have to dodge rocks that are falling from the sky. You can play with up to 2 players and the goal is to survive as long as possible.
